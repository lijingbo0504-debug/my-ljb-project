## 四、 部署与安装步骤

> ⚠️ **注意：** 禅道官方 Linux 一键安装包（zbox）默认内置了 Apache、PHP 和 MariaDB，建议将其解压到 /opt 目录下，切勿解压到其他路径，否则服务无法正常启动。

### Step 1：下载禅道一键安装包

使用 wget 工具从官方镜像源下载 Linux 64位开源版安装包：

```bash
# 进入存放安装包的临时目录 cd /usr/local/src # 下载官方安装包（若速度较慢，可提前下载好通过 SFTP 上传） wget https://dl.cnezsoft.com/zentao/18.12/ZenTaoPMS.18.12.zbox_amd64.tar.gz
```

### Step 2：解压并安装

将下载好的压缩包直接解压到 /opt 目录：
```bash
# 使用 -C 参数指定解压到 /opt 目录
tar -zxvf ZenTaoPMS.18.12.zbox_amd64.tar.gz -C /opt/
```

### Step 3：配置服务端口（可选）

为了避免与系统中现有的 Apache (80) 或 MySQL (3306) 端口冲突，建议修改默认端口（此处将 Web 端口修改为 8080，数据库端口修改为 3307）：

```bash
# 修改禅道 Web 服务端口为 8080
/opt/zbox/zbox -ap 8080

# 修改禅道内置数据库端口为 3307
/opt/zbox/zbox -mp 3307
```


### Step 4：启动禅道服务

执行内置脚本启动服务，并设置开机自启：
```bash
# 启动 zbox 服务
/opt/zbox/zbox start

# 将启动命令写入开机自启脚本中
echo "/opt/zbox/zbox start" >> /etc/rc.local
chmod +x /etc/rc.d/rc.local
```


### Step 5：防火墙放行

若系统启用了 firewalld，需要开放对应的网页访问端口：
```bash
# 开放 8080 端口
firewall-cmd --zone=public --add-port=8080/tcp --permanent

# 重新加载防火墙配置
firewall-cmd --reload
```

## 六、 故障排查与 FAQ

> 💡 **要求：** 请记录至少一个在部署过程中遇到的实际问题及解决方法。

- **问题描述：** 服务启动成功，但在宿主机浏览器输入 http://IP:8080 提示“连接超时”或“无法访问此网站”。
    
- **排查思路：**
    
    1. 在服务器本地使用 curl [http://127.0.0.1:8080](http://127.0.0.1:8080) 发现有 HTML 源码返回，证明服务本身正常。
        
    2. 检查网络连通性，宿主机可以 ping 通服务器 IP。
        
    3. 怀疑是 Linux 防火墙拦截。
        
- **解决方案：** 执行 systemctl status firewalld 发现防火墙处于 Active 状态。通过执行 firewall-cmd --zone=public --add-port=8080/tcp --permanent 开放端口并刷新防火墙后，问题解决。
